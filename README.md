# 使用说明
## 快速启动

    $./main.py --pc [题目编号] --tc [测试用例编号]
    
    举例：
    ./main.py --pc 171 --tc 2
    ************************************************************
                              第171道题
    ************************************************************
    测试用例：2, 结果：通过，耗时: 2.15μs

_注意：_
1. 不允许只加参数--tc 
2. 如果不加参数，则遍历测试所有题目与测试用例
3. 如果只加参数--pc，则遍历该题目的所有测试用例
